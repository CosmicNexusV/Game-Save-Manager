# Game Save Manager

基于 Docker 的游戏存档管理 Web 应用，支持存档上传/下载/备注、游戏本体压缩包管理、EXE 图标自动提取。

## 快速开始

```bash
docker-compose up -d
```

浏览器访问 `http://<设备IP>:8085`

## 密码保护

访问页面需要输入密码（无需账户名）。

- **密码文件**：`app/password.txt`（容器首次启动时自动生成）
- **初始密码**：`123456`
- 若 `app/password.txt` 已存在，启动时不会覆盖，请妥善保管

**修改密码**：登录后点击右上角「设置」→「修改密码」

**重置密码**：删除 `app/password.txt` 后重启容器，将重新生成初始密码 `123456`

```bash
rm app/password.txt
docker-compose restart
```

## 数据目录

| 路径（宿主机）              | 说明           |
| --------------------------- | -------------- |
| `/mnt/nas/game-save-data/`  | 游戏存档与数据 |
| `app/password.txt`          | 访问密码（哈希存储）|

## 功能

- 游戏列表管理（添加/编辑/删除）
- 自动从 `.exe` 提取图标，或手动上传图片
- 存档上传（支持文件夹）、下载（ZIP 打包）、删除
- 可配置每个游戏最多保留的存档数量（超出自动清理旧存档）
- 游戏本体压缩包上传与下载
- 时间以东八区（UTC+8）显示

## 端口

| 宿主机端口 | 容器端口 |
| ---------- | -------- |
| 8085       | 5000     |

## 重新构建

修改代码后重新构建镜像：

```bash
docker-compose up -d --build
```
